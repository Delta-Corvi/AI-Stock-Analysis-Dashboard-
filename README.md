# 🚀 AI Stock Analysis Dashboard

An advanced stock market analysis application with AI-powered predictions and interactive web interface for comprehensive financial insights.

## 📋 Overview

This project combines traditional technical analysis with AI capabilities to provide comprehensive financial market insights. The application uses machine learning for price predictions and Google Gemini for intelligent textual analysis.

## ✨ Key Features

### 📊 Advanced Technical Analysis
- **Interactive Candlestick Charts**: OHLC visualization with dynamic trend lines
- **Technical Indicators**: RSI, MACD, EMA with professional visualizations
- **Support & Resistance Lines**: Automatic calculation based on local maxima and minima

### 🔮 AI-Powered Forecasting
- **15-Day Price Forecasting**: Price predictions using machine learning models
- **Confidence Intervals**: 95% probability bands to assess uncertainty
- **Multiple Models**: Linear regression with polynomial features and time series analysis

### 🤖 Intelligent Text Analysis
- **Detailed Financial Reports**: Comprehensive analysis generated by Google Gemini
- **Dividend Analysis**: Yield evaluation and distribution policy assessment
- **Risk Assessment**: Risk profile and volatility evaluation
- **Technical Analysis Narrative**: Textual interpretation of technical indicators

### 🎨 User Interface
- **Modern Design**: Responsive UI with gradients and glassmorphism effects
- **Interactive Dashboard**: Intuitive controls and real-time visualizations
- **Pre-configured Examples**: Templates to quickly start analysis

## 🛠️ System Architecture

### Project Structure
```
├── app_gradio.py              # Main web interface
├── agents/
│   ├── finance_agent.py       # Agent for financial analysis
│   ├── analysis_agent.py      # Agent for charts and predictions
│   └── agent_tools.py         # Shared tools
├── charting/
│   └── charts.py             # Chart generation and ML predictions
├── data/
│   └── data_provider.py      # Data management with caching
└── gemini_summarizer.py      # Google Gemini integration
```

### Core Components

#### 📈 Charting System
- Uses **Plotly** for high-quality interactive charts
- Trend line calculation algorithms based on **scipy**
- Predictive models with **scikit-learn**

#### 🤖 AI Agents
- **FinanceAgent**: Generates textual analysis using Gemini
- **AnalysisAgent**: Produces visualizations and numerical predictions

#### 💾 Data Management
- **Intelligent Caching**: Cache system with 24-hour TTL
- **YFinance Integration**: Real-time market data
- **Robust Error Handling**: Fallbacks and comprehensive logging

## 🚀 Installation & Configuration

### Prerequisites
- Python 3.8+
- Google Cloud account with Gemini API enabled

### 1. Clone Repository
```bash
git clone https://github.com/yourusername/ai-stock-analysis-dashboard.git
cd ai-stock-analysis-dashboard
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Environment Variables Setup
Create a `.env` file in the project root:
```env
GEMINI_API_KEY=your_gemini_api_key_here
# or
GOOGLE_API_KEY=your_google_api_key_here
```

### 4. Run Application
```bash
python app_gradio.py
```

The application will be available at `http://localhost:7860`

## 🎯 How to Use

### Basic Analysis
1. **Enter Ticker Symbol**: e.g., AAPL, MSFT, TSLA
2. **Select Time Period**: From 1 month to maximum available data
3. **Choose Analysis Type**:
   - 📊 Detailed Financial Report
   - 💰 Dividend Analysis
   - 📈 Technical Analysis
   - 🔍 Risk Assessment
4. **Enable/Disable Forecasting**: Toggle for 15-day predictions

### Interpreting Results

#### Candlestick Charts
- **Green/Red Candles**: Up/down trading days
- **Dashed Lines**: Support and resistance trends
- **Orange Area**: Prediction zone with confidence interval

#### Technical Indicators
- **RSI**: Values >70 indicate overbought, <30 oversold
- **MACD**: Signal line crossover indicates trend changes

#### AI Predictions
- **Orange Line**: Median forecast
- **Shaded Area**: 95% confidence interval
- **Volatility Range**: Key levels to monitor

## ⚙️ Advanced Configuration

### Model Customization
You can modify predictive model parameters in `charting/charts.py`:
- `lookback_period`: Period for trend line calculation
- `forecast_days`: Prediction days (default: 15)
- `confidence_level`: Confidence interval level

### Cache Optimization
Modify cache duration in `data/data_provider.py`:
```python
# Change from 24 hours to desired value
if datetime.datetime.now() - file_creation_time < datetime.timedelta(hours=24):
```

## 📊 Usage Examples

### Tesla (TSLA) Analysis
```python
# Programmatic example
from agents.finance_agent import FinanceAgent
from agents.analysis_agent import AnalysisAgent

finance_agent = FinanceAgent()
analysis_agent = AnalysisAgent()

# Complete analysis
summary = finance_agent.run("Complete technical analysis", "TSLA")
charts = analysis_agent.get_stock_charts("TSLA", "1y", include_forecast=True)
```

### Batch Analysis
To analyze multiple stocks, iterate through a list:
```python
tickers = ["AAPL", "MSFT", "GOOGL", "AMZN"]
results = {}

for ticker in tickers:
    results[ticker] = finance_agent.run("Financial report", ticker)
```
